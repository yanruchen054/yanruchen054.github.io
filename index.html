Research

Wearable Acetone Gas Sensor for Measuring Exhaled Breath of Diabetes Patients ACS Applied Materials & Interfaces, Oct 2024

Semiconductor metal oxide (SMO) gas sensors are low‐cost, real‐time, and long‐lasting but usually need elevated temperatures. We developed a flexible, room‐temperature sensor using UV‐assisted design by embedding ZnO nanoparticles into a PDMS porous framework. An innovative low‐modulus island‐bridge structure allowed 60% stretchability.

Enhanced Conductivity in Laser‐Induced Graphene–Silicon Carbide Laminated Nanosheets for Flexible Strain Sensors and Pulse Wave Velocity Assessment IEEE MEMS Conference 2024 (0115)

Novel flexible strain sensor based on LIG–SiC laminated nanosheets with strong stability and accuracy in PWV tests.

Perfusion Heterogeneity Assessment in Human CT Data Using PM3 Platforms and Machine Learning Segmentation IEEE EMBC 2020 Conference (2037)

Texture analysis using improved fractal dimension metrics for CT perfusion imaging.

Quantifying Microvascular Alterations Due to a Pharmacological Agent IEEE EMBC 2020 Conference (2044)

Analyzed microvascular network changes induced by CORM2 using intravital microscopy and network extraction.

Dynamic Color‐Changing Solar Cell Based on Pr₂O₃ Characteristics Jun 2019 – Jun 2020

Photochromic solar cell with dynamic color‐changing ability without consuming its own generated energy.
